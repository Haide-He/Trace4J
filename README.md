# Trace4J

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.17181411.svg)](https://doi.org/10.5281/zenodo.17181411)
[![Documentation Status](https://app.readthedocs.org/projects/trace4j/badge/?version=latest)](https://trace4j.readthedocs.io/en/latest/?badge=latest)


Trace4J is a lightweight, flexible, and insightful performance tracing tool for Java, designed to bridge the gap between Java source code and its execution behavior on modern CPUs.

## Quick Start

```bash
git clone https://github.com/Haide-He/Trace4J.git && cd Trace4J

# Installation
make

# Setup environment variables
export Trace4J_HOME=/path/to/Trace4J
export PATH=$Trace4J_HOME/bin:$PATH
export LD_LIBRARY_PATH=$Trace4J_HOME/lib:$LD_LIBRARY_PATH
export PRECISE_IP=1
```
